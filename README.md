## Como baixar o projeto 📑

```bash

    # Clonar o repositório
    $ git clone htpps://github.com/pecraveiro/rocketmusic

    # Entrar no diretório
    $ cd rocketmusic

    #Instalar as dependências
    $ yarn install
    
    #Iniciar o projeto
    $ yarn start
```
