## Como baixar o projeto 📑

```bash

    # Clonar o repositório
    $ git clone https://github.com/pecraveiro/CPP-Tutorial-for-Beginners-Full-Course

    # Entrar no diretório
    $ cd rocketmusic

    #Instalar as dependências
    $ yarn install
    
    #Iniciar o projeto
    $ yarn start
```
